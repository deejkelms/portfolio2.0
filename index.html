Welcome to my web portfolio. My name is Derek and I’m a junior full-stack web developer currently coding in Fort Lauderdale, FL. If there’s one thing to know about me, it’s that I love technology (and cars and Batman, but that’s a story for another time).

I taught myself HTML in high school and have continued learning different languages ever since. Frontend development

Whether you need a new website or updates to an existing application, I would definitely love to help.


I am learning to write code that conforms to best practices and standards. I use technologies like HTML5, CSS3 and jQuery to create beautiful, interactive and fully functional interfaces. I build using a responsive grid system with frontend frameworks such as HTML5 Boilerplate, Bootstrap and Pure to ensure that layouts are now quicker, smarter, and up to date.
